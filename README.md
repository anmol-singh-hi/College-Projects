**Salient Features of the Software:**

Content Intelligence: Analyzes video transcripts using Large Language Models to support rapid information absorption.

Granularity Control: Allows users to define the depth of the summary (Short, Medium, or Long) to match their specific learning needs.

Multimodal Output: Provides both a structured Markdown text report and a high-fidelity audio narration.

Human-in-the-Loop Design: Acts as a decision-support tool, allowing users to guide the AI with custom prompts to focus on specific technical or thematic details.

Explainable Synthesis: Prioritizes transparency by using clear, context-aware summaries that reference the source material accurately.

Audio Accessibility: Generates human-readable, spoken narrations using the gTTS integration to support on-the-go learning.

Reactive Architecture: Utilizes a Streamlit frontend for a seamless, responsive interface with real-time token streaming.
